WeatherDE - Weather based on German Zip code

Based on the Module WeatherUS
Credits: Jason Schaitel, Jason_Schaitel@hotmail.com
         Jakob Hansen, hansen3000@hotmail.com

Another Appleseed desktop module - more to download on http://www.Appleseedportal.net


INSTALL 
1. Go to Admin all and to add module definition. 
2. Point to install.xml install file
3. Add the module to a page
4. Edit module settings: 
5. Use it! 8) 

Note: You may add a style definition for the LABEL tag to get a smaller labels on the edit page.

HISTORY
WeatherUS Ver. 1.0 - 5.  dec 2002 - Moved from original IBS into Appleseed by Jakob Hansen
WeatherDE Ver. 1.0 - 21. apr 2003 - Adapted for use with wetter.com and German Zip Codes by Mario Hartmann

ISSUES AND KNOWN PROBLEMS:
- Tested with Appleseed version 1.2.8.1634


MODULE SETTINGS
---------------
zip code   - the zip code of the city.
city Index - if there are more than one city with the given zip
			 or your city is not named on the Image, 
			 you can choose the different (known) cities by index.
design     - choose your preferred design.
setting    - choose forcast or today.
