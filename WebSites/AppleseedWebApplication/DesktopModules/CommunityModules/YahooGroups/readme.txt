One File Module Example - Yahoo Groups
Credits: Jakob Hansen, hansen3000@hotmail.com, Rahul Singh, anantatman@hotmail.com


Another Appleseed desktop module - more to download on http://www.Appleseedportal.net


Note: Before you install the "OneFileModuleKit" must have been installed!

INSTALL
1. Copy file YahooGroups.ascx to folder Appleseed\DesktopModules
   Note: Do not add YahooGroups.ascx to the project - no compiling needed!
2. Run SetupDBPatch.bat (or execute DBPatch.sql in SQL Query Analyzer)
3. Log on as Admin and add the "YahooGroups (OneFileModule)" module to a tabpage 
4. Edit module settings: 
     enter "YahooGroupName=Appleseedportal;" in field "Settings string"
5. Use it! ;o)
