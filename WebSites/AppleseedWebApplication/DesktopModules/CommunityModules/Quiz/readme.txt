Quiz
In this module a XML file holds all questions.


Credits: Oliver Draskovic, www.smarttrain.de


Another Appleseed desktop module - more to download on http://www.Appleseedportal.net


INSTALL
1. Go to Admin all and to add module definition. 
2. Point to install.xml install file
3. Add the module to a page
4. Edit module settings: See below
5. Use it! ;o) 
Note: The module is automatically installed when you install Appleseed.
The install procedure is only required if you deleted the module in Admin all

Module settings
---------------
QuizName: A textstring that describes the quiz. Default value is "Australia (Demo1)"
XMLsrc: XML files that holds the quiz questions. Default value is "/Quiz/Demo1.xml"


IMPORTANT : Quiz folder "\Quiz" has to be open for writing for user ASPNET becuase 
            Pie graph is saved there after quiz is finished. The Folder is placed in:
            "Appleseed\_Appleseed" (and in "Appleseed\_New portal" if you created new portals)
